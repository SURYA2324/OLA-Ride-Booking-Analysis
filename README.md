# OLA-Ride-Booking-Analysis
OLA Ride Booking Analysis project built using SQL, Excel, and Power BI. The dashboard provides insights into booking trends, cancellations, vehicle demand, customer behavior, and operational performance through interactive visualizations and data driven analysis.
